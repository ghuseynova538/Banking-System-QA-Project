# 🏦 Banking System QA Portfolios

## 📌 Layihə Haqqında
Bu layihə, rəqəmsal bankçılıq platformalarında (Giriş və Pul Köçürmələri) funksional və təhlükəsizlik testlərinin aparılması üçün nəzərdə tutulub.

## 🧪 Test Ssenariləri (Core Coverage)
* **SQL Testing:** Backend tərəfdə məlumatların düzgünlüyünün yoxlanılması.
* **Security Checks:** SQL Injection və Brute-force hücumlarına qarşı müdafiə ssenariləri.
* **API Validation:** Postman vasitəsilə status kodlarının və JSON cavablarının doğrulanması.

## 🦈 Git Workflow & Achievements
Layihə daxilində **Pull Request (PR)** iş axını tətbiq olunub və GitHub tərəfindən **Pull Shark** nişanı ilə təltif edilib. Bu, komanda mühitində kodun nəzərdən keçirilməsi (Code Review) bacarığını simulyasiya edir.
