# 🏦 Banking System Quality Assurance Project
This project showcases a professional end-to-end testing lifecycle for a digital banking application.

## 🎯 Project Goal
To validate the security, data integrity, and functional reliability of the Core Banking modules (Login, Fund Transfer, Account Management).

## 📂 Project Structure
* **Documentation/**: Master Test Plan, Test Strategy, and Risk Assessment.
* **Test_Cases/**: Detailed functional and edge-case scenarios.
* **SQL_Validation/**: Scripts for backend data verification and integrity checks.
* **Defects/**: Professional bug reports with technical evidence.

## 🛠 Tech Stack & Environment
* **Platform:** iOS 18.6.2 (Testing environment)
* **Hardware:** HP Windows 10 pro
* **Tools:** GitHub, SQL, Postman, 3uTools
